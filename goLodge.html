GoLodge

GoLodge is a full-stack app where lounge owners list their lounges and travelers to check information about available listed lounges. I built this app from scratch, with Bootstrap, Express and MongoDB. This app took a lot of heavy lifting to create User Authentication and Authorization, and to display the lounges on a map.
LIVEGITHUB
Project Overview
GoLodge let's users create a lodge by allowing them input the lodge name, the price of the lodge, additional details and information about the lodge, and the location of the lodge.
Lodge owners are also allowed to add pictures of the lodge they want to list.
Users/Travellers can view all information about a listed lodge, and most importantly they can add rating and reviews to any lodge they have visited. Lodge Owners can delete and modify lodges they list.
Key Features
  • You add images to your lodge using Image Upload.
  • You can add a Star Rating and Text Review to any Lodge.
  • Authentication & Authorization; you cannot create a lodge if you're not logged in or registered & only the creator of a lodge can alter the lodge.
  • In the details page, location of a lodge is displayed on a map.
  • You can edit the information of a lodge you created.
  • You can edit the images you have uploaded for a particuler lodge.
  • There is form validation, that makes sure you input the correct details while creating a lodge.
  • One the homepage,there is a cluster map that show the location of all the listed lodges.
  • Error and Success messages are flashed back to users.
